lightbulb Key Design Principles

Ultra-Light Base

#f0f0f7 background + pure white cards = depth without darkness

Purple-Tinted Grays

Ink colors use purple undertones for refined, warm appearance

Low-Opacity Borders

12-22% rgba borders create gentle separations that work everywhere

Muted Accent Blue

#5781a9 is intentionally desaturated for professional calm

Blue-Tinted Shadows

Shadows use rgba(79,70,229) to tie into the accent color system

Fast Micro-Interactions

140-180ms transitions create responsive, immediate feeling
